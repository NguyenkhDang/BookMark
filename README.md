A simple bookmark manager build with Vite, Node.js + Express as backend being deploy on Render
<h1>Features</h1>

- Add bookmarks with URL and name
- Search bookmarks by name (live search)
- Delete bookmarks
- Fully connected frontend + backend
- Deployed backend on Render
- Environment variable configuration for backend API

<h1>Technologies Used</h1>

- **Frontend:** React, TypeScript, Vite, CSS Modules
- **Backend:** Node.js, Express, TypeScript
- **Other:** CORS, Fetch API
- **Deployment:** Render
<h1>Intitial Setup</h1>
  Back End
  
```
  1. cd Back-End
  2. npm install
  3. npm run dev
```

  Front End
```
  1. cd Front-End
  2. npm install
  2. npm run dev
```
